# Custom YOLO Object Detection

Interactive object detection application using YOLOv8 and Streamlit. Upload an image and detect multiple objects with bounding boxes and confidence scores. The app allows users to customize the YOLO model and detection confidence according to their needs.

## Features

- Supports YOLOv8 models: `yolov8n`, `yolov8s`, `yolov8m`
- Adjustable confidence threshold for detection
- Displays original and detected images side by side
- Shows detected object details including ID, type, and confidence
- Fully customizable settings to match user/device requirements
- User-friendly interface using Streamlit
